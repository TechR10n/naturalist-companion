# Use Case - Appalachia I-81 Corridor

## Scenario

A traveler drives along I-81 through the Shenandoah Valley and wants a quick, grounded guide to what they are seeing.

## Goals

- Provide a short list of stops spaced along the corridor.
- Explain visible geology features in plain language.
- Offer citations for every factual claim.

## Flow

1. User selects a route or corridor.
2. System samples points along the route.
3. Wikipedia GeoSearch returns candidate pages.
4. The guide presents 5-12 stops with citations.

## Success criteria

- Each stop card has 2-4 citations.
- The guide is readable in under 5 minutes.
- Stops are spaced and not clustered.
